**Переменная** - это область памяти, в которой хранятся наши данные. В переменную задается имя и значение, например:

```python
test = 1
value = "2"
result = 3.0
test2 = True
```

Каждая переменная содержит тип данных, например: `float`, `int`, `str`, `bool` и т.д.
